# ADR / Archive

## Назначение

Архив ADR, формально признанных устаревшими или заменёнными другим
решением (superseded/deprecated).

## Что здесь хранится

- ADR, перенесённые сюда по явному решению, с пометкой причины и ссылкой
  на заменяющий документ.

## Что запрещено хранить здесь

- Действующие ADR — см. [../](../).
- Автоматически перенесённые документы: перенос в архив выполняется
  только осознанно, как часть отдельного решения, а не этим этапом.
