# ADR — Architecture Decision Records

## Назначение

Действующие архитектурные решения проекта VistaRoom AI: что решено,
почему, и какие альтернативы были отклонены. Это раздел про "почему
решили именно так" (для описания текущего состояния архитектуры см.
[../architecture/](../architecture/)).

Этот README не заменяет и не изменяет существующий
[ADR_INDEX.md](ADR_INDEX.md) (реестр решений) и [ADR_MAP.md](ADR_MAP.md)
(навигационная карта) — оба документа остаются главными точками входа в
систему ADR.

## Что здесь хранится

- Действующие ADR-документы (`ADR-XXX-*.md`).
- [ADR_INDEX.md](ADR_INDEX.md) — реестр решений.
- [ADR_MAP.md](ADR_MAP.md) — карта архитектурных областей.
- [templates/](templates/) — шаблоны для новых ADR.
- [archive/](archive/) — устаревшие/заменённые ADR.

## Что запрещено хранить здесь

- Черновики решений до утверждения — см.
  [../governance/records/](../governance/records/).
- Roadmap и governance-контент общего характера — см.
  [../governance/](../governance/).
