# ADR / Templates

## Назначение

Шаблоны для создания новых ADR. Канонический шаблон находится в
[../../templates/ADR_TEMPLATE.md](../../templates/ADR_TEMPLATE.md) — эта
папка предназначена для ADR-специфичных вариантов шаблона или чеклистов,
если они понадобятся в будущем.

## Что здесь хранится

- ADR-специфичные вариации шаблона (при необходимости).

## Что запрещено хранить здесь

- Заполненные ADR — см. [../](../).
