# References / Providers

## Назначение

Справочная информация о внешних провайдерах (AI-модели, API-вендоры и
т.п.): возможности, ограничения, ценообразование.

## Что здесь хранится

- Справочные материалы по провайдерам, используемым или рассматриваемым
  платформой.

## Что запрещено хранить здесь

- Решения о выборе провайдера — это ADR, см. [../../adr/](../../adr/)
  (например, терминология провайдеров уже зафиксирована в ADR-001).
