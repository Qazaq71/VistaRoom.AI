# References

## Назначение

Справочные материалы, не являющиеся собственной документацией проектных
решений/архитектуры: результаты исследований, анализ конкурентов,
справочная информация о провайдерах, внешние стандарты.

## Что здесь хранится

- [research/](research/) — заметки и материалы исследований.
- [competitors/](competitors/) — анализ конкурентов.
- [providers/](providers/) — справочная информация о провайдерах (AI/API
  и т.п.).
- [standards/](standards/) — внешние стандарты и спецификации.

## Что запрещено хранить здесь

- Собственные ADR/PCS/ACS проекта — они выражают решения проекта, а не
  внешние справочные материалы.
