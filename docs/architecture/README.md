# Architecture

## Назначение

Описания текущего состояния архитектуры платформы VistaRoom AI: обзорные
документы, диаграммы, снимки на ключевых вехах и результаты архитектурных
аудитов. Это раздел про "как устроено сейчас", а не про "почему решили
именно так" (для решений см. [../adr/](../adr/)).

## Что здесь хранится

- Обзорные документы архитектуры верхнего уровня.
- [diagrams/](diagrams/) — архитектурные диаграммы.
- [milestones/](milestones/) — снимки архитектуры на ключевых вехах.
- [audits/](audits/) — отчёты архитектурных аудитов и ревью.

## Что запрещено хранить здесь

- ADR (архитектурные решения) — см. [../adr/](../adr/).
- Roadmap-документы — см. [../roadmap/](../roadmap/).
- PCS/ACS-спецификации платформы — см. [../platform/](../platform/).
