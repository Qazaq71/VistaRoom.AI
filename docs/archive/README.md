# Archive

## Назначение

Общий архив устаревших документов проекта, не относящихся к конкретному
специализированному архиву (для ADR — [../adr/archive/](../adr/archive/),
для PCS/ACS — [../platform/pcs/archive/](../platform/pcs/archive/) и
[../platform/acs/archive/](../platform/acs/archive/)).

## Что здесь хранится

- [deprecated/](deprecated/) — депрецированные документы общего
  назначения.

## Что запрещено хранить здесь

- Активные документы любого типа.
- Автоматически перенесённые документы: перенос в архив выполняется
  только по явному решению, а не автоматически.
