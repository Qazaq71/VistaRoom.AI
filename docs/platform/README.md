# Platform

## Назначение

Платформенная документация: спецификации возможностей платформы (PCS,
ACS), knowledge packs и справочные описания реализованных capability.

## Что здесь хранится

- [pcs/](pcs/) — Platform Capability Specifications.
- [acs/](acs/) — Agent/AI Capability Specifications.
- [knowledge-packs/](knowledge-packs/) — структурированные пакеты знаний
  для AI-контекста.
- [capabilities/](capabilities/) — справочные описания реализованных
  возможностей платформы.

## Что запрещено хранить здесь

- Runtime-код (`.ts`/`.tsx`) — этот раздел только про документацию.
- ADR — см. [../adr/](../adr/).
- Roadmap-контент — см. [../roadmap/](../roadmap/).
