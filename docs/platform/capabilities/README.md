# Platform / Capabilities

## Назначение

Справочные описания реализованных возможностей платформы — что уже
работает, в отличие от спецификаций (PCS/ACS), которые описывают, что
должно быть реализовано.

## Что здесь хранится

- Описания реализованных capability-модулей платформы.

## Что запрещено хранить здесь

- PCS/ACS-спецификации — см. [../pcs/](../pcs/) и [../acs/](../acs/).
- Runtime-код.
