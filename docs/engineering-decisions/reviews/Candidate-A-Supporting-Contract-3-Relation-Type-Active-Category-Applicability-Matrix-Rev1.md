## 6. Complete Relation Type Г— Residential-34 matrix

### 6.1 Structural basis for the matrix (not a per-cell restatement)

Contract 2 Annex C fixes the general endpoint applicability ceiling for
each Relation Type using only node-type predicates (`c2.pred.001`вҖ“
`c2.pred.009`) вҖ” `Room`, `StructuralElement`, `Object`, and
`FreeSpaceRegion` вҖ” none of which reference any room category identity.
Contract 1 Rev19 establishes all 34 entries as active residential
category identities but does not require Contract 3 to assert that
every instance contains any particular StructuralElement, Object,
opening, floor, ceiling, path, or clearance endpoint. Existence and
eligibility of concrete endpoints remain instance-level evidentiary
questions governed by Contract 2.

No accepted upstream artifact introduces a category-level exclusion for
Adjacency, Containment, or Blocking among the Residential-34
identities. Consequently, the category-level applicability
determination is uniform across all 34 categories for all three
relation types,
**including Blocking**. Blocking is category-applicable for all
Residential-34 categories without exception (`c3.rule.001`, В§7.1).
Assertion of a concrete Blocking relation for any specific RoomCase
instance remains entirely instance-dependent and continues to require
satisfaction of Contract 2's endpoint predicates, Blocking-mode
predicates (Annex F), evidence, confidence, and provenance rules вҖ”
none of which this Contract narrows, widens, or substitutes at the
category level (MAJOR-1 closure). Whether a given RoomCase instance of
a small enclosed category (e.g. `pantry`, `dressing_room`,
`walk_in_closet`) in fact contains a walkable path or clearance-relevant
endpoint is an instance-level evidentiary question resolved entirely
under Contract 2; the absence of such evidence for a particular
instance produces no positive Blocking record for that instance вҖ” it
does not change, and this Contract does not use it to change, the
category's own applicability disposition.

FreeSpaceRegion-involving endpoint participation in Adjacency,
Containment and Blocking is uniformly reserved for every category at
the Contract-2 level (В§3.1). This Contract does not restate that
reservation as 34 additional `reserved` matrix rows and does not
represent it as a Contract-3-owned disposition (В§5, MINOR-2 closure);
it remains an imported, category-independent Contract 2 fact.
