# 4. Decision Group 3 — Track and Module Model

## OD-09 — Strategic Track and Major Module Separation

Project Owner подтверждает:

```text
Strategic Track ≠ Major Module
```

Strategic Track является долгосрочным направлением Roadmap.

Major Module является отдельно bounded capability с собственным lifecycle.

## OD-10 — Multiple Modules per Track

Project Owner подтверждает, что один Strategic Track может включать несколько Major Modules.

Major Module также может зависеть от нескольких Strategic Tracks.

## OD-11 — Track A Is Not Closed by One Module

Project Owner подтверждает, что закрытие текущего bounded Major Module:

```text
Bounded Room Understanding / Spatial Perception
```

не означает автоматического закрытия всего:

```text
Track A — Spatial Perception
```

## OD-12 — No Lifecycle Status for Strategic Tracks

Project Owner подтверждает, что Strategic Track сам по себе не получает lifecycle-status:

```text
Primary Active Module
```

Этот статус применяется только к конкретному bounded Major Module.
