# Governance

## Назначение

Процессы принятия решений и управление проектом на уровне выше отдельного
ADR: методология принятия решений, оценка инициатив, ревью процессов,
формальные записи решений.

## Что здесь хранится

- [decision-playbook/](decision-playbook/) — методология и правила
  принятия решений.
- [scorecards/](scorecards/) — оценки инициатив/решений.
- [reviews/](reviews/) — ревью governance-процессов.
- [records/](records/) — формальные записи принятых решений.

## Что запрещено хранить здесь

- Технические архитектурные решения (ADR) — см. [../adr/](../adr/).
- Roadmap-контент — см. [../roadmap/](../roadmap/).
