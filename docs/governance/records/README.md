# Governance / Records

## Назначение

Формальные записи принятых решений (Decision Records) на уровне проекта
— не архитектурных (см. [../../adr/](../../adr/)), а организационных и
процессных.

## Что здесь хранится

- Заполненные decision records, оформленные по шаблону
  [../../templates/DECISION_RECORD_TEMPLATE.md](../../templates/DECISION_RECORD_TEMPLATE.md).

## Что запрещено хранить здесь

- Архитектурные решения (ADR) — см. [../../adr/](../../adr/).
