# Governance / Decision Playbook

## Назначение

Методология и правила принятия решений: критерии оценки, процессы
эскалации, роли и ответственность.

## Что здесь хранится

- Playbook-документы, описывающие процесс принятия решений.
- Критерии и рамки оценки инициатив.

## Что запрещено хранить здесь

- Конкретные записи принятых решений — см. [../records/](../records/).
- Заполненные scorecards — см. [../scorecards/](../scorecards/).
