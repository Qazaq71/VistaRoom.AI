# Implementation Packages

Implementation Packages содержат инструкции по реализации уже принятых архитектурных решений.

Они:

- не являются ADR;
- не являются ED;
- не изменяют ACS;
- не изменяют PCS;
- не изменяют Gate Specifications;

Их назначение — быть связующим слоем между governance-документами и реализацией в Claude Code.

## Current Packages

- ADR-005 Integration — Implementation Package v1.0

В дальнейшем новые Implementation Packages должны добавляться в этот список.
