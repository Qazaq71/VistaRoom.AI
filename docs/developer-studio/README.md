# Developer Studio

## Назначение

Документация внутренних инструментов разработки и анализа платформы
(Developer Studio): бенчмарки, prompt lab, эксперименты, дашборды и
инструменты-исследователи архитектуры/знаний/решений.

## Что здесь хранится

- [benchmark/](benchmark/) — документация по бенчмаркам моделей/промптов.
- [prompt-lab/](prompt-lab/) — эксперименты и заметки по prompt engineering.
- [experiments/](experiments/) — протоколы и результаты экспериментов.
- [dashboards/](dashboards/) — описания дашбордов и метрик.
- [architecture-explorer/](architecture-explorer/) — инструмент
  исследования архитектуры.
- [knowledge-explorer/](knowledge-explorer/) — инструмент исследования
  knowledge packs/баз знаний.
- [decision-dashboard/](decision-dashboard/) — дашборд обзора governance
  решений.

## Что запрещено хранить здесь

- Runtime-код самих инструментов (см. `src/`).
- Platform-спецификации — см. [../platform/](../platform/).
