# Engines

Слой интеллектуальной логики Developer Studio.

В отличие от [Services](../services/README.md), которые выполняют действия
и интеграции (чтение файлов, сохранение данных, работа с API, логирование,
стоимость, изображения), Engines содержат логику принятия решений и обработку
данных — то, что определяет результат, а не то, как он сохраняется или
передаётся.

## Будущие engines

- **PromptEngine** — построение и оптимизация промптов
- **StyleEngine** — логика подбора и применения стилей
- **RoomAnalyzer** — анализ комнаты на изображении
- **MaterialEngine** — логика подбора материалов
- **FurnitureEngine** — логика подбора и расстановки мебели
- **LightingEngine** — логика анализа и настройки освещения
- **ObjectDetection** — распознавание объектов на изображении
- **BenchmarkEngine** — логика оценки и сравнения результатов генерации
- **AutomaticMasks** — автоматическое построение масок

Пока engines не реализованы — папка является архитектурной заготовкой.
